2  Tests for One Continuous Population Mean

Learning Objectives

By the end of this chapter, you should be able to:

  • Formulate hypotheses for testing a single population mean or proportion.

  • Choose and conduct an appropriate one-sample t-test or one-sample z-test based on study context and assumptions.

  • Differentiate between cases where the population variance is known and unknown, and select the appropriate test for each case.

  • Perform one-sample t-tests and z-tests using both R and Python, and interpret the outputs.

  • Apply statistical reasoning to real-world data examples, including evaluating assumptions and limitations.

This chapter introduces statistical tests for analyzing a single sample, a common task in data analysis across many disciplines. These methods are used when we want to compare a sample-based estimate to a known or hypothesized population value. For example, we may want to test whether the average recovery time after a treatment differs from a clinical standard, whether student test scores exceed a benchmark, or whether the proportion of successes in a group differs from an expected rate. In each case, hypothesis testing helps us assess whether the observed difference is large enough to provide evidence against the null hypothesis, or whether it could reasonably be explained by random sampling variation.

In this chapter, we review tests for making inference about a single population parameter based on one sample. We focus on two common settings:

  • A population mean, where the response variable is continuous, and
  • A population proportion, where the response variable is binary.

For a single population mean, the choice of test depends on whether the population variance is known or unknown. When the population variance is unknown, we use the one-sample t-test. When the population variance is known, we use the one-sample z-test. For binary outcomes, we use a one-sample proportion z-test to compare the population proportion to a hypothesized value.

The choice of test depends on the type of response variable, the information available about the population variance, and the sample size. Each section below introduces the test setup, hypotheses, assumptions, theoretical background, and implementation in both R and Python using real or simulated data.

2.1 One-sample t-test for the mean

2.1.1 Review

Use this test when:

  • You have one sample from a population.
  • The response variable is quantitative (also known as continuous).
  • You want to test whether the population mean is equal to a specific reference value.
  • The population variance is unknown.
  • The sample is approximately normal, or the sample size is large enough for the sampling distribution of the sample mean to be approximately normal.

In this section, we focus on the case where the population variance is unknown. The known-variance case will be discussed later using the one-sample z-test.

The one-sample t-test is used when we want to compare a sample mean to a hypothesized population mean. Suppose we observe a random sample of size \(n\),

\[ X_1, X_2, \dots, X_n, \]

from a population with unknown mean \(\mu\). We want to assess whether the data provide enough evidence to conclude that \(\mu\) differs from a specified reference value, denoted by \(\mu_0\).

The main statistical question is:

Is the population mean statistically different from the hypothesized value \(\mu_0\)?

For example, if the hypothesized value is \(\mu_0 = 200\), then the question becomes:

Does the sample provide evidence that the population mean is different from 200?

Because the population variance is unknown, we estimate it using the sample standard deviation. This additional uncertainty is why the test statistic follows a t-distribution rather than a standard normal distribution.

2.1.2 Study design

Now that we have reviewed the general idea behind the one-sample t-test, let us apply it to a real data example. In this section, we use the penguins data set from the palmerpenguins package in R. This data set contains measurements for penguins observed near Palmer Station, Antarctica. Some of the main quantitative variables include flipper_length_mm, which records the length of a penguin’s flipper in millimetres; bill_length_mm, which records the length of the penguin’s bill in millimetres; bill_depth_mm, which records the depth or thickness of the bill in millimetres; and body_mass_g, which records the penguin’s body mass in grams.

For this example, the response variable of interest is flipper_length_mm, which records penguin flipper length in millimetres. We treat the observed penguins in the data set as a sample from a broader population of similar penguins. Let \(\mu\) denotes the population mean flipper length of penguins. Suppose we want to compare the population mean flipper length to a reference value of 200 mm. In other words, we want to assess whether the sample provides evidence that the average flipper length in the population is different from 200 mm.

The main statistical question is:

Does the data provide evidence that the population mean flipper length of penguins is different from 200 mm?

In this example, the hypothesized population mean is

\[ \mu_0 = 200. \]

To keep the exploratory and inferential parts of the analysis separate, we will split the data into a training set and a test set. The training set will be used for exploratory data analysis, and the test set will be used to conduct the formal hypothesis test.

2.1.5 Testing Settings

Before writing the hypotheses, we summarize the testing setup for this example. The response variable is flipper_length_mm, which is a quantitative variable measured in millimetres. The parameter of interest is the population mean flipper length, denoted by \(\mu\).

In this example, we compare the population mean flipper length to the reference value

\[ \mu_0 = 200. \]

Since the population standard deviation is unknown, we use the sample standard deviation to estimate it. Therefore, the appropriate test is a one-sample t-test.

The formal test will be conducted using the test set. This keeps the exploratory analysis and the hypothesis test separate: the training set was used to understand the data, while the test set is used to calculate the test statistic and p-value.

For this test, we use:

  • Response variable: flipper_length_mm
  • Parameter of interest: \(\mu\), the population mean flipper length
  • Reference value: \(\mu_0 = 200\)
  • Test: one-sample t-test
  • Population variance: unknown
  • Sample used for inference: test_set
  • Significance level: \(\alpha = 0.05\)

Because this is a one-sample t-test, we assume that the observations are independent and that the sampling distribution of the sample mean is approximately normal. The exploratory analysis suggested that the raw flipper length distribution is not perfectly normal, partly because penguin species differ in their typical flipper lengths. However, the sample size is reasonably large and there are no severe outliers, so using the one-sample t-test is still a reasonable approach for this example.

2.1.6 Hypothesis Definitions

We now translate the research question into formal statistical hypotheses. Recall that \(\mu\) denotes the population mean flipper length of penguins, and the reference value is

\[ \mu_0 = 200. \]

Since the research question asks whether the population mean flipper length is different from 200 mm, we use a two-sided test.

The hypotheses are:

\[H_0: \mu = 200 \quad vs \quad H_a: \mu \neq 200\]

The null hypothesis states that the population mean flipper length is 200 mm. Under this hypothesis, any difference between the sample mean and 200 mm is attributed to random sampling variation.

The alternative hypothesis states that the population mean flipper length is not 200 mm. This includes both possibilities: the population mean could be less than 200 mm or greater than 200 mm.

In words, we are testing whether the test-set data provide sufficient evidence that the average flipper length in the broader penguin population represented by these data differs from 200 mm.

2.1.7 Test Flavour and Components

The one-sample t-test compares the sample mean to the hypothesized population mean. In this example, the hypothesized value is \(\mu_0 = 200\) mm. Let \(\bar{x}\) denotes the sample mean flipper length in the test set, and let \(s\) denotes the sample standard deviation of flipper length in the test set. If the test set contains \(n\) penguins, the test statistic is

\[ t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}. \]

In this formula:

  • \(\bar{x}\) is the sample mean flipper length.
  • \(\mu_0\) is the hypothesized population mean, which is 200 mm in this example.
  • \(s\) is the sample standard deviation.
  • \(n\) is the sample size.
  • \(s / \sqrt{n}\) is the estimated standard error of the sample mean.

The numerator, \(\bar{x} - \mu_0\), measures how far the sample mean is from the hypothesized value. The denominator, \(s / \sqrt{n}\), scales this difference by the estimated variability of the sample mean. Therefore, the test statistic measures how many estimated standard errors the sample mean is away from the hypothesized value.

Because the population standard deviation is unknown, we estimate it using the sample standard deviation \(s\). Under the null hypothesis, \(H_0: \mu = 200\), the test statistic follows a t-distribution with \(n-1\) degrees of freedom. Since our alternative hypothesis is two-sided, i.e. \(H_A: \mu \neq 200\) the p-value measures the probability of observing a test statistic at least as extreme as the one obtained from the data, in either direction, assuming that the null hypothesis is true.

At the significance level \(\alpha = 0.05\), we reject the null hypothesis if the p-value is less than 0.05. Otherwise, we do not reject the null hypothesis.

2.1.8 Inferential Conclusions

The goal of the one-sample t-test is to decide whether the observed sample mean is far enough from the hypothesized value to provide evidence against the null hypothesis. In this example, the null and alternative hypotheses are:

\[ H_0: \mu = 200 \quad \textit{vs} \quad H_a: \mu \neq 200 \] where \(\mu\) denotes the population mean flipper length of penguins. The hypothesized value is 200 mm. After computing the test statistic and p-value, we compare the p-value to the significance level \(\alpha = 0.05\).

If the p-value is less than 0.05, we reject the null hypothesis. In that case, we conclude that the data provide sufficient evidence that the population mean flipper length is different from 200 mm.

If the p-value is greater than or equal to 0.05, we do not reject the null hypothesis. In that case, we conclude that the data do not provide sufficient evidence that the population mean flipper length is different from 200 mm.

It is important to phrase the second conclusion carefully. Failing to reject the null hypothesis does not prove that the population mean is exactly 200 mm. It only means that, based on the available sample, we do not have enough evidence to conclude that it differs from 200 mm.

Based on these data, an average flipper length of 200 mm is a plausible value for the broader penguin population represented by this sample.

In plain language, the observed average flipper length is not significantly different from 200 mm, and the small difference between 200.3 mm and 200 mm can reasonably be explained by random sampling variation.

2.2 One-sample z-test for the mean

2.2.1 Review

We use this test when: - The population variance σ² is known, and - The sample comes from a normally distributed population, or the sample size is large (typically ( n )).

The test statistic is:

\[ z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}} \]

Where: - ( \(\bar{x}\) ) is the sample mean
- ( \(\mu_0\) ) is the hypothesized population mean
- ( \(\sigma\) ) is the known population standard deviation
- ( n ) is the sample size

We compare the calculated ( z )-value to a standard normal distribution to compute a p-value or make a decision based on a critical value.

Let’s revisit the earlier scenario: you want to assess whether a new method of teaching introductory physics at the University of British Columbia (UBC) improves student performance compared to historical results. This time, assume that the population standard deviation is known from previous years of large-scale data collection. This allows us to use a z-test. Suppose the average physics score in the population is: $$

$$

\[ \mu \quad \text{(mean physics score at UBC)} \] And we collect a random sample of \(n\) students, recording their scores:

\[ X1,X2,…,XnX_1, X_2, \dots, X_n \]

The Key question becomes:

Is the sample mean statistically different from a known reference value?

If, for example, the historical average physics score is known to be 75, then our question becomes more specific:

Is the mean physics score in the sample statistically different from 75?

2.3 One-sample z-test for proportions

2.3.1 Review

Use this test when: - The variable is binary (success/failure, yes/no, etc.), and - You want to test a population proportion ( p ), using a large enough sample.

The test statistic is:

\[z = \frac{\hat{p} - p_0}{\sqrt{p_0(1 - p_0)/n}}\]

Where: - ( \(\hat{p}\) ) is the sample proportion
- ( \(p_0\) ) is the hypothesized population proportion
- ( \(n\) ) is the sample size

Just like the one-sample t-test is used to assess differences in means, the one-sample z-test is used when: • You are working with proportions (e.g., fraction of penguins with a specific characteristic), • The sample size is sufficiently large to justify using the normal approximation. This test is ideal for binary outcomes: for example, whether a penguin is of the Adelie species or not. Suppose you want to test whether the proportion of Adelie penguins in your sample differs from a historically established value. Let’s say historical records suggest that 45% of penguins in a certain region were Adelie. You want to know whether the proportion in your current sample is statistically different from 0.45.

The central question becomes: > ** Is the proportion of Adelie penguins in the sample different from 0.45?? We are again working with the Palmer Station Penguins** dataset.
